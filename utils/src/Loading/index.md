## Loading 组件

这是一个Loading组件，也是一个示例，根据这个例子可以编写全局组件文档

代码演示:

```tsx
/**
 * title: 基本使用
 * defaultShowCode: true
 */
import { Loading } from 'microup/utils';

export default () => (
  <>
    <Loading />
  </>
);
```
## API

| 属性          | 说明         | 类型   | 默认值 |
| ------------- | ------------ | ------ | ------ |
| 参数        | 参数  | object | {name:'loading'} |
| 参数2       | 参数2 | object | {name:'loading'} |
